# PromptCache UI Directions

## Shared UI Themes

Across all directions:

- left navigation for core product areas
- strong search entry point
- primary split between library and workshop
- dense but calm information layout
- keyboard-forward interactions for heavy users

## Direction 1: Studio

This direction treats the app like a creative tool for serious builders.

### Feel

- editorial-meets-developer
- layered panels
- warm neutrals with sharp accent color
- high-contrast type hierarchy

### Core Layout

- Left rail for navigation
- Center workspace that changes by section
- Right contextual panel for metadata, AI suggestions, and linked records

### Best For

- users who spend lots of time refining prompts
- a premium workshop feeling
- side-by-side comparisons and revision flows

### Signature Screens

- Workshop with editor in the center and critique cards on the right
- Library with strong search and card/list toggle
- Session map with connected prompt cards and response chips

## Direction 2: Mission Control

This direction leans into operations, traceability, and system thinking.

### Feel

- technical
- structured
- dashboard-inspired
- slightly futuristic but still usable

### Core Layout

- Left nav
- Top command/search bar
- Main grid with reusable panels

### Best For

- users who care about sessions, lineage, and integrations
- clear views of prompt runs and outputs
- heavier information density

### Signature Screens

- Session timeline with linked prompt runs
- Search page with advanced filters and semantic result grouping
- Integrations page showing MCP/plugin health and capabilities

## Direction 3: Card Catalog

This direction focuses on memory, retrieval, and collecting reusable prompt assets.

### Feel

- tactile and library-like
- elegant card system
- calmer visual rhythm
- a little less technical

### Core Layout

- Navigation + large search-first home
- prompt cards with visible tags and snippets
- workshop opens as a focused canvas from the library

### Best For

- users whose main need is finding and reusing prompt assets quickly
- a knowledge-base-first experience

### Signature Screens

- Search-centric home page
- Rich prompt detail page with versions and linked sessions
- Saved response gallery under each prompt

## Recommended Direction

Direction 1, Studio, is the strongest fit for v1.

Why:

- It best supports the "prompt workshop" idea.
- It gives us a premium editor-first experience without losing library/search power.
- It can absorb sessions and integrations naturally through side panels and linked views.

## Suggested Visual System For Studio

- Typeface mix: a distinctive serif or humanist display for page titles plus a clean sans for UI and editor chrome
- Palette: parchment, ink, muted graphite, with one vivid signal color like vermilion or electric teal
- Surfaces: textured gradients or subtle paper-like depth rather than flat panels
- Motion: staggered panel reveals, prompt diff transitions, session-link trace animations

## Key Screens To Design First

1. Home dashboard
2. Prompt library
3. Workshop editor
4. Session detail view
5. Prompt detail with versions and responses
6. Integrations/settings

## Suggested UX Patterns

- Command palette for search and navigation
- Slash commands in editor for prompt actions
- One-click "improve", "critique", and "branch" actions
- Prompt lineage view for versions and related prompts
- Save response directly from workshop runs
