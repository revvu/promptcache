# PromptCache

PromptCache is a prompt library and workshop for writing, improving, storing, searching, and organizing prompts used in development workflows.

## Current Status

The repository now contains:

- product docs in `docs/`
- a polished Next.js app shell in `src/`
- Supabase schema scaffolding in `supabase/schema.sql`
- environment setup in `.env.example`

## Stack

- Next.js 14
- TypeScript
- Tailwind CSS
- Supabase

## Getting Started

1. Install dependencies with `npm install`
2. Copy `.env.example` to `.env.local`
3. Fill in your Supabase and OpenAI keys
4. Run `npm run dev`
